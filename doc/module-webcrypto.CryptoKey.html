Creates a new CryptoKey object.
Parameters:
Name Type Description

extractable boolean Indicating if the raw information may be exported or not.
algorithm AlgorithmIdentifier Represents a cipher the key has to be used with.
usages Array.<string> Array of values indicating what the key can be used for.
