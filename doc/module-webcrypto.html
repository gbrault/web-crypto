Module: webcrypto

The webcrypto module provides access to all cryptographic methods.
It is the equivalent to the Crypto interface.

(static) getRandomValues(typedArray)

Generates cryptographically random values. The array given as the parameter is filled with random numbers (random in its cryptographic meaning).
Parameters:
Name Type Description
typedArray Int8Array | Uint8Array | Uint16Array | Int32Array | Uint32Array An integer-based TypedArray which is filled with random numbers. All elements in the array are going to be overridden.
